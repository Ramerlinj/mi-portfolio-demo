# Mi portafolio (demo)

Landing page sencilla de portafolio hecha con HTML y CSS. Incluye secciones de hero, habilidades y proyectos con un estilo oscuro moderno.

## Estructura

- `index.html`: estructura del sitio.
- `style.css`: estilos principales y variables de color.

## Como ver el sitio

1. Abre `index.html` en tu navegador.
2. (Opcional) Usa una extension como Live Server para recarga automatica.

## Personalizacion rapida

- Cambia el nombre y descripcion en la seccion hero.
- Reemplaza las imagenes de proyectos por tus capturas.
- Ajusta colores en `:root` dentro de `style.css`.

## Vista previa

El sitio usa una paleta oscura con acentos azules y una grilla de proyectos.
